今天学习的模型是双指针/二分模型。

遇到有序数组时，可以考虑使用这两种模型，当然变幻后的有序也需要考虑一下，比如 977.

共性就是找到不变量，找到分界点，然后一点点收缩要处理的数据。

边界就是能否分割成几个区域：
- 未扫描区
- 已处理区/确认区
- 淘汰区/无用区

使用快慢指针作为区域分界线，根据题目的条件变换指针边界，一步步将处理结果归纳到已处理区。

还有一个点是未扫描区是未来的，扫描顺序需要按照过去-> 未来的顺序，比如844的#代表左删除，如果从左往右扫描，那么需要知道#才可以确认前一个字符是否删除（可以用栈）。
如果从右向左扫描，那么就可以遇到#时就可以确认需要跳过几个字符再继续扫描，也不需要读到那个字符到底是什么。

### 复习

#### 二分
二分的步骤：
- 求mid值
- 条件对比mid
- 缩进

初始化left，right后，while的条件如何确定呢？
- while left < right
  - 区间缩小到一个点，这个点再根据要求进行处理，一般处理找某个点。
- while left <= right
  - 区间缩小到一个点后，继续计算这个点和target的条件。一般找分界线时候用。

对于left，right的缩进条件，要具体分析，比如 right = mid - 1, left = mid + 1
缩进时，要看mid是否需要包含进去，包含进去的比如找边界，\[1,3,4,4,4,5,6],如果mid=3时，target=4，此轮不清楚mid前后是什么，所以mid要包进来，在\[0..3]继续
寻找左边界。这时right = mid;
如果是找单一值时，mid一般会直接出结果。

#### 指针
使用双指针需要找不变量，确定题目的规律
1.[0..slow] 确定，(slow, fast)无效数据，这样就一步一步进行slow替换.用于操作数组
2.区间长度固定，就像拉一条线一起前进，用来处理倒数问题。
3.求解区间长度，就像拉一根弹簧，fast先前进，符合条件后，slow前进来确定最短长度。
4.无效区间规则，动态区间长度，需要找到区间外的点，（844）
5.求解区间长度，就像拉一根弹簧，fast先前进，符合条件后,slow前进来确定最大长度
